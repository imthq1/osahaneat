# osahaneat

## Mô Tả Dự Án
Dự án `osahaneat` là một ứng dụng **full-stack** bao gồm **frontend** (React) và **backend** (Spring Boot).

# Technical

## Backend:

1. Framework: Spring Boot
2. Database: MySQL
3. Cloudinary
4. VNPAY
5. RabbitMQ
6. CRUD
7. Oauth2
## Frontend:

1. Framework: React
2. Language: TypeScript
3. UI Library: Ant Design
4. Styling: SCSS
